# Sales Showcase UI Prototype

Tablet-first frontend prototype for a customer-facing digital catalogue.

## Stack

- React 19
- Vite 8
- TypeScript
- Tailwind CSS v4

## Implemented in this setup pass

- New React + Vite workspace scaffolded
- Tailwind CSS v4 integrated via Vite plugin
- First-pass UI implementation based on the provided design attachment:
  - Entry screen
  - Category screen
  - Catalogue viewer screen
  - More info modal mockup
  - Loading/empty/error/success feedback states

Note: This pass focuses on UI structure and styling only. API wiring, routing, state management, and realtime sync are next steps.

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
